## Stack Tecnologico

```
Frontend:     React 18 + Vite + TailwindCSS
Backend:      Node.js + Express + Sequelize
CMS:          Ghost 5.x
Database:     MySQL 8
Container:    Docker + Docker Compose
CI/CD:        GitHub Actions
Proxy:        Nginx
```
